# digit-recognizer-computer-vision
Based on the Kaggle dataset from digit-recognizer (https://www.kaggle.com/competitions/digit-recognizer), use different computer vision Neural Network methods to properly classify each handwritten image. Solve multi-classification problem. 

**IMPORTANT NOTE: to run notebook, make sure to have your Kaggle API key setup**

## Approach

### Convolutional Neural Network (CNN)
- Convolution layers for spatial feature extraction
- Pooling layers for dimensionality reduction
- Strong performance on image classification tasks

### Regularization Techniques
- Dropout
- Batch normalization
- Early stopping

### Optimization Methods
- Adam optimizer
- Stochastic Gradient Descent (SGD)
- Learning rate tuning

**Required libraries will be in the `requirements.txt`**

## Evaluation Metrics

This is a multi-class classification problem, we'll use categorization accuracy of the predictions. 
- Accuracy
- Confusion Matrix
